# Calendar App 

## client:
- ReactJs
- React Modal
- FullCalendar
- axios
- moment
- react-datetime

## server:
- body-parser
- dotenv
- express
- moment
- mongoose (local MongoDB)
- nodemon

## Further instructions

In server directory, create .env with:

`MONGODB_URI=mongodb://localhost:27017/calendar-app`

run `npm start` in server directory

run `npm start` in client directory
